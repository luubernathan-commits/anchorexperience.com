## Overview

This repository contains the static marketing website for [ANCHOR Experience](https://anchorexperience.com). The site showcases our retreat design services, the 3 Pillars Assessment, and introduces visitors to our experiential philosophy.

## Features

- Responsive design with dark mode support
- 3 Pillars interactive assessment
- Retreat showcase and service overview
- Optimised for performance on Cloudflare's edge network

## Tech Stack

- **HTML5 / CSS3** — Semantic markup with custom styling
- **Cloudflare Pages** — Global CDN deployment
- **Custom Design System** — Teal/cream palette, Zodiak & Satoshi fonts
